Dynamic programming examples

Edit Distance

The edit distance between two strings is the minimum steps needed change a source string into a destination string by; inserting a character into the source strings, deleting a character in the destination string; changing a single charatcer in both strings that is in the same location.

Knapsack Problem 1/0

The Knapsack problem asks for a given set of items with each item having a cost factor (like weight) and a value factor indicating how important or valued this item is how to maximize the value of a subset of items all the while limiting the total cost to a certain maximum amount.

Longest Common Subsequence

The longest common subsequence between two strings is the longest string of characters shared by both strings in the same order. The output shows how the lcs strings is layed out in both strings. A star (*) means the characte is ignored, a space is used to align both strings.

Coin Change Problem

The Coin Change problem asks for how to give change for a certain amount using a certain set of coin denominations; each set has a potential unlimited quantity to use.
