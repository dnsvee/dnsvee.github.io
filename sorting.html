On this page a selection of sorting algorithms. For each algorithm the source code is displayed and the last section of the webpage displays the result of a performance test by sorting a random array of numbers.

Swap sort

Scan the entire array and repeatedly swap two elements when they are out of order. Stop when after scanning the entire array no swap was needed.

Quick sort

Choose an appropriate value for the array that splits it in halves of roughly equal size. Create three new arrays; one with all the values smaller than the pivot; one with al the values equal value to the pivot; and one with all the values larger. Run quick sort on the resulting halves with smaller/larger values. Combine the three arrays when done.

Merge sort

First step is to recursively divide an array in two halves and keep dividing them until each resulting halve contains at most one element. Then merge the resulting in arrays of increasing size until all of them are merged into one array. Merging two sorted arrays into one sorted array can be done with reasonable efficiency.

Selection sort

Split the array into two halves. First part is the array that is sorted; second halve is the unsorted part. On each iteration of the loop select the smallest element from the remaining part of the array that's still unsorted and insert into the sorted part of array.

Insertion sort

Split the array into two parts. First part is sorted. Second part unsorted. On each iteration select the first element from the unsorted part and repeatedly swap it down into the correct place.

Heap sort

A binary heap is an array that contains a binary tree. A parent node of index i has child nodes at location i * 2 and i * 2 + 1. A max. heap has at it's root the maximum value of the array. Take an unsorted array and turn it into a heap by ensuring the parent/child relationship holds for each node. By repeatedly popping off this max. value and placing it in the correct place at the end of the array you can sort the unsorted array. On each step the heap becomes one smaller and the sorted array one larger.
